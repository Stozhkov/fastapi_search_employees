This is a test project with FastAPI framework.

API urls:

/search_employees/ method GET
available query parameters:
* name (string),
* email (string),
* age_min (integer),
* age_max (integer),
* company (string),
* join_date_start (datetime string),
* join_date_stop (datetime string),
* job_title (string),
* gender (string),
* salary_min (integer),
* salary_max (integer).
